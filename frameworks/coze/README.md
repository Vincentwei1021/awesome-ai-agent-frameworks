# 🤖 Coze（扣子）— 字节跳动 AI Bot 搭建平台

> [国内版](https://www.coze.cn/) | [海外版](https://www.coze.com/) | 类型: SaaS 平台

## 一句话概述

Coze 是字节跳动的**零代码 AI Bot 搭建平台**，通过可视化界面配置 Bot 的人设、技能、工作流和知识库，最快 5 分钟上线一个 AI 应用。

## 🚀 快速上手

### 创建一个 AI 助手（5 分钟）

1. **登录** [coze.cn](https://www.coze.cn/)
2. **创建 Bot** → 填写名称和描述
3. **配置人设**：
   ```
   你是一个 Python 编程助教，面向零基础学员。
   回答问题时：
   - 先用通俗的话解释概念
   - 然后给出代码示例
   - 最后提一个练习题
   ```
4. **添加技能**：搜索插件 + 代码执行插件
5. **添加知识库**：上传 Python 教材 PDF
6. **测试** → **发布**到飞书 / 微信

### 工作流示例

```
[用户提问] → [意图识别] → 
    ├─ 编程问题 → [知识库检索] → [LLM 回答]
    ├─ 运行代码 → [代码执行插件] → [返回结果]
    └─ 其他 → [通用回答]
```

## ✅ 优势

1. **零代码** — 纯界面操作，5 分钟上手
2. **插件丰富** — 500+ 官方插件，覆盖主流需求
3. **免费额度** — 个人使用基本免费
4. **一键发布** — 直接发布到飞书/微信/网页
5. **字节基建** — 稳定性和性能有保障
6. **多模态** — 原生支持图片/音频理解

## ❌ 劣势

1. **不开源** — 纯 SaaS，无法自部署
2. **灵活性有限** — 复杂逻辑受平台约束
3. **数据安全** — 数据存储在字节服务器
4. **生态锁定** — 深度使用后迁移成本高
5. **模型选择** — 国内版只能用豆包模型

## 🎯 适用场景

| 适合 | 不适合 |
|------|--------|
| 快速搭建客服 Bot | 需要私有化部署 |
| 个人 AI 助手 | 需要复杂多 Agent 协作 |
| 企业内部飞书 Bot | 数据安全要求极高 |
| 教育/培训场景 | 需要深度定制 |
| 非技术团队 | 开发者需要代码级控制 |
